# 🚀 DevOpsGPT — AI-Powered DevOps Automation Platform

DevOpsGPT is a full-stack AI-powered DevOps automation platform that dynamically analyzes uploaded source code and GitHub repositories to generate production-ready infrastructure configurations.

## 🌐 Live Demo

Frontend:
https://dev-ops-gpt-blush.vercel.app/

Backend API:
https://devopsgpt-in3d.onrender.com/docs

## ⚡ Local Setup

### Clone Repository

```bash
git clone https://github.com/sathvik2903/DevOpsGPT.git
cd DevOpsGPT
```

---

### Backend Setup

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

Backend runs on:

```bash
http://127.0.0.1:8000
```

---

### Frontend Setup

```bash
cd devopsgpt-frontend
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:3000
```

---

## 🔑 Environment Variables

Create `.env` files:

### Backend

```env
COHERE_API_KEY=your_api_key
```

### Frontend

```env
NEXT_PUBLIC_API_URL=your_backend_url
```
